10. arti dari kata te amo?​


Aku mencintaimu.

Pembahasan

¡Te amo! (aku mencintaimu) adalah salah satu ungkapan yang sering kita dengar dalam bahasa Spanyol. Te amo adalah ungkapan untuk orang yang disayangi, seperti pacar atau pasangan.

Selain ¡te amo! adapun “¡te quiero!”, frasa ini biasanya digunakan ungkapan sayang untuk orang tua, teman atau sahabat.

____

Kata kerja infinitif (dasar) yang digunakan pada “¡te amo!” adalah “amar”, yang artinya “mencintai”. Kata kerja “amar” berubah menjadi “amo” apabila bertemu subjek pertama yaitu “yo/saya”.

Jikalau subjek bertemu kata kerja maka terjadi konjugasi, namun subjeknya harus dihilangkan. Contohnya :

¡Yo te amo! = ¡Te amo! (Karena Yo/saya dan amo/mencintai, maka "yo" bisa dihilangkan).

Berikut konjugasi kata kerja "amar".

yo → amo. (dari amar menjadi amo).tú → amas. (dari amar menjadi amas).él/ella → ama. (dari amar menjadi ama).nosotros/as → amamos. (dari amar menjadi amamos).vosotros/as → amáis. (dari amar menjadi amáis).ellos/ellas → aman. (dari amar menjadi aman).

15. kata kerja"nemasu"jika diubah ke dalam bentuk -te menjadi​


Kata kerja 寝ます (Nemasu) jika diubah ke dalam bentuk Te menjadi​ 寝て (Nete), karena merupakan kata kerja kelompok 2 (Ichidan-doushi), jadi hanya perlu mengubah bentuk akhirannya saja.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

動詞 (Doushi) / Kata Kerja

Dibagi menjadi 3 kelompok, yakni :

1. 五段動詞 (Godan-doushi)

=> Disebut juga KK Kelompok 1, merupakan kelompok kata kerja yang akhiran bunyi dari kata nya merupakan salah satu dari U, Tsu, Ru, Mu, Nu, Bu, Ku, Gu, atau Su.

2. 一段動詞 (Ichidan-doushi)

=> Disebut juga KK Kelompok 2, merupakan kelompok kata kerja yang akhiran bunyi dari kata nya antara Iru atau Eru. Tetapi ada juga beberapa kata kerja yang berakhiran bunyi tersebut tapi termasuk ke Godan.

3. 不規則動詞 (Fukisoku-doushi)

=> Disebut juga KK Kelompok 3, merupakan kelompok kata kerja tidak beraturan yang hanya terdiri dari Suru dan Kuru saja.

Dalam Bahasa Jepang, terdapat yang namanya konjugasi kata kerja atau yang biasa disebut dengan 動詞の活用 (Doushi no katsuyou). Contoh bentuk konjugasi yang akan saya bahas kali ini adalah bentuk Te.

______________________________

Pembahasan :

て形 (Te-kei) / Bentuk Te

Merupakan salah satu bentuk konjugasi dalam Bahasa Jepang. Bentuk ini sangat sering digunakan dalam Bahasa Jepang.

Fungsi KK Bentuk Te

Kalau pada kata kerja, bentuk Te ini memiliki fungsi antara lain :

Untuk menghubungkan kata atau kalimat secara berurutanUntuk membuat kata perintah atau permintaan yang sopanUntuk menunjukkan keterangan caraUntuk menyatakan sebab-akibat

Cara Mengubah KK ke Bentuk Te

Caranya berbeda-beda untuk tiap kelompok kata kerja, sebagai berikut.

1. Untuk Godan-doushi / KK Kelompok 1

=> Dengan mengubah akhiran :

U, Tsu, Ru menjadi TteMu, Nu, Bu menjadi NdeKu menjadi IteGu menjadi IdeSu menjadi Shιte

Contohnya seperti Kau menjadi Katte, Nomu menjadi Nonde, Aruku menjadi Aruite, Oyogu menjadi Oyoide, dan Mawasu menjadi Mawashιte.

2. Untuk Ichidan-doushi / KK Kelompok 2

=> Dengan mengubah akhiran :

Ru menjadi Te

Contohnya seperti Miru menjadi Mite, dan Taberu menjadi Tabete.

3. Untuk Fukisoku-doushi / KK Kelompok 3

=> Dengan mengubah :

Suru menjadi ShιteKuru menjadi Kite

Contohnya seperti Benkyou suru menjadi Benkyou shιte, dan Modotte kuru menjadi Modotte kite.

- - - - - - -

Nemasu merupakan bentuk Masu atau bentuk sopan dari kata kerja Neru, yang artinya adalah "Tidur". Kata ini berakhiran bunyi 'Eru', jadi termasuk ke dalam Ichidan-doushi atau KK kelompok 2. Maka dari itu, untuk mengubahnya ke bentuk Te adalah dengan mengubah akhiran Masu atau Ru pada kata tersebut menjadi Te. Nemasu atau Neru menjadi Nete.

32. Apa perbedaan antara akhiran kata kerja U dan TE, seperti 帰る dan 帰って yang artinya sama² pulang.​


Perbedaan antara 帰る (Kaeru) dengan 帰って (Kaette) adalah Kaeru merupakan kata kerja bentuk kamus, dan memiliki arti "Pulang" saja, sedangkan Kaette merupakan kata kerja bentuk Te, dan memiliki arti "Pulang", "Pulang, lalu...", atau "Pulanglah!".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

Doushi「動詞・どうし」

Merupakan istilah umum untuk "Kata kerja" dalam Bahasa Jepang. Kata kerja dalam Bahasa Jepang dibagi menjadi beberapa kelompok. Kata kerja kelompok 1 atau Godan Doushi, yang berisi kata kerja yang berakhiran bunyi U, Tsu, Ru, Mu, Nu, Bu, Ku, Gu, dan Su. Kata kerja kelompok 2 atau Ichidan Doushi, yang berisi kata kerja yang berakhiran bunyi Iru dan Eru. Dan terakhir adalah kata kerja kelompok 3 atau Fukisoku Doushi, yang berisi dua kata kerja tidak beraturan yaitu Suru dan Kuru.

Kata kerja Bahasa Jepang memiliki berbagai macam bentuk. Untuk kata kerja yang berakhiran bunyi yang saya sebutkan di atas, itu termasuk kata kerja bentuk kamus. Adapun bentuk-bentuk kata kerja lainnya seperti bentuk Masu, bentuk Te, bentuk Ta, dan lain-lain.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

KK Bentuk Kamus / Jisho-kei「辞書形」

Kata kerja bentuk ini adalah kata kerja yang dituliskan dalam kamus, maka dari itu disebut bentuk kamus. Kata kerja bentuk kamus akhiran bunyi nya selalu bunyi U. Bentuk kamus ini tidak memiliki fungsi khusus, tapi kata kerja bentuk kamus itu kesannya kasual, jadi biasa digunakan di percakapan informal saja. Selain itu bentuk kamus juga biasa digunakan saat kata kerja nya ada di awal atau tengah-tengah kalimat.

Contoh kalimat :

1. うちに帰る

= Uchi ni kaeru

= Pulang ke rumah

2. 来るなら教えてね

= Kuru nara oshiete ne

= Kalau datang kasih tau ya

- - - - - - -

KK Bentuk Te / Te-kei「て形」

Kata kerja bentuk ini adalah kata kerja yang akhiran bunyi nya Te atau De. Fungsi dari bentuk ini sendiri ada banyak, antara lain sebagai berikut.

Membuat kata sambungMembuat kata perintah halusMenunjukkan keterangan caraMenunjukkan keterangan alat

Untuk mengubah kata kerja ke bentuk Te ini, caranya berbeda untuk setiap kelompok kata kerja. Caranya tidak akan saya jelaskan di sini, kalian bisa melihat salah satu link yang saya berikan di bawah.

Contoh kalimat :

1. うちに帰って、寝ます

= Uchi ni kaette, nemasu

= Pulang ke rumah, lalu tidur

2. ねえ、これ見て！

= Nee, kore mite!

= Hey, lihatlah ini!

3. 心を込めて歌う

= Kokoro wo komete utau

= Menyanyi dengan sepenuh hati

4. 眼鏡をかけてテレビを見る

= Megane wo kakete terebi wo miru

= Nonton TV menggunakan kacamata

38. kapan dan bagaimana pengunaan kata s'il vous plait dan s'il te plait​


Grammaire française

S'il te/vous plait merupakan akhiran kalimat yang menyatakan permohonan atau permintaan. Bedanya, s'il vous plait digunakan dalam situasi formal, sedangkan s'il te plait digunakan dalam situasi informal. Konstruksi kalimatnya :

Kt. kerja (terkonjugasi) + s'il te/vous plait
