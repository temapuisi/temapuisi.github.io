4. tepat jam 12.00 siang lonceng gereja berbunyi dengan frekuensi 500 HZ, jika periode getaran 20 detik berapa getaran yang di hasilkan​


Getaran

Diketahui :

f = frekuensi = 500 Hz t = waktu = 20 s

Ditanya :

n = getaran ?

Jawaban

f = n/t

n = f × t

n = 500 × 20

n = 10.000 kali ✔

GETARAN

f = 500 Hz

t = 20 s

n = ... ?

n = f · t

n = 500 · 20

n = 10.000 getaran ✔

Bab KPK dan FPB
Matematika SD Kelas V

15 = 3 x 5
20 = 2² x 5
24 = 2³ x 3

kpk = 2³ x 3 x 5
       = 8 x 15
       = 120 menit
       = 2 jam

24.00 + 2 jam = 00.00 + 2.00
                       = 2.00

jadi, berdentang bersama lagi pada jam 2.00 dini hari
