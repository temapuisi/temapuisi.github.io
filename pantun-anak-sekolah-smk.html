Perbandingan senilaiPerbandingan berbalik nilai

Pada umumnya perbandingan senilai dapat dirumuskan ; X₁ / X₂ = Y₁ / Y₂

Pada umumnya perbandingan berbalik nilai dapat dirumuskan ; X₁ / X₂ = Y₂ / Y₁
