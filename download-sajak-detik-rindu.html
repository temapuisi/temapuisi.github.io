30. pak danang mendownload sebuah vidio yang berukuran 50mb dengan kecepatan download internet 100kbps, berapa detikkah waktu yang dibutuhkan pak danang untuk mendownload vidio tersebut?


Bila dikonversikan: 1 mega byte = 1.024 kilo byte
maka 50 mega byte = 51.200 kilo byte
kecepatan download = 100 kilo byte per second
maka waktu yang dibutuhkan: ukuran file / kecepatan download = 51.200 / 100 =512 second.

Jadi waktu yang dibutuhkan pak danang adalah 512 detik (kira-kira 8 menit)

Seorang gamer mendownload file sebesar 81 GB, dengan kecepatan download sebesar 45 Mbps, Ketika download sudah mencapai 79%, maka speed download berubah menjadi 35 Mbps, dan stabil sampai file selesai didownload. - v0 = 45mb/s

t0 = 1422 s

= 1422+486

= 1908 s

Jawaban:

9 jam 33 menit 20 detik

Penjelasan:

Ketika speed 50Mbps

75% x 120GB = 90GB

90GB X 8bit = 720Gb

720Gb = 720000Mb

720000Mb/50Mbps = 14400s

Ketika speed 12Mbps

120GB-90GB= 30GB

30GB X 8bit = 240Gb

240Gb = 240000Mb

240000Mb/12Mbps = 20000s

Total Waktu Download

14400s+20000s = 34400s

34400s = 9 jam 33 menit 20 detik
