Perpangkatan adalah sebuah bentuk operasi dalam ilmu matematika yang boleh digunakan apabila sebuah bilangan dikali dengan bilangan yang sama lebih dari satu kali. Pangkat sendiri merupakan banyaknya bilangan yang dikali dengan bilangan yang sama. Letak pangkat sendiri berada diatas bilangan yang dikalikan tersebut.

Bilangan berpangkat merupakan perkalian berulang suatu bilangan, dimana bilangan dapat berpangkat bulat positif, nol, maupun bulat negatif. Secara sederhana penulisan bilangan jenis ini adalah sebagai berikut :

[tex]\bf \blue{ {a}^{n} = a \times a \times a... \times a}[/tex]

a disebut bilangan pokok atau basis, sedangkan n disebut pangkat atau eksponen

_____________________

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 4

3² = ( 3 x 3 ) = 9

4² = ( 4 x 4 ) = 16

5² = ( 5 x 5 ) = 25

[tex]\:[/tex]

_____________________

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 8

3³ = ( 3 x 3 x 3 ) = 27

4³ = ( 4 x 4 x 4 ) = 64

5³ = ( 5 x 5 x 5 ) = 125

[tex]\:[/tex]

_____________________

Contoh Perpangkatan 4:

2⁴ = ( 2 x 2 x 2 x 2 ) = 16

3⁴ = ( 3 x 3 x 3 x 3 ) = 91

4⁴ = ( 4 x 4 x 4 x 4 ) = 256

5⁴ = ( 5 x 5 x 5 x 5 ) = 625

[tex]\:[/tex]

_____________________

Contoh Perpangkatan 5:

2⁵ = ( 2 x 2 x 2 x 2 x 2 ) = 32

3⁵ = ( 3 x 3 x 3 x 3 x 3 ) = 273

4⁵ = ( 4 x 4 x 4 x 4 x 4 ) = 1.024

5⁵ = ( 5 x 5 x 5 x 5 x 5 ) = 3.125
