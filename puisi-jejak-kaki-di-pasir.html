12. Tiga orang petani pergi ke sawah melewati ruas jalan yang lembek. Ternyata kedalaman jejak kaki petani C lebih dalam dari jejak kaki petani B dan jejak kaki B lebih dalam dari jejak kaki A. Jika luas penampang kaki mereka sama , maka perbandingan berat ketiga petani tersebut adalah …. *


Petani yang terdiri dari 3 orang yaitu petani A, B dan C pergi ke sawah melewati ruas jalan yang lembek. Luas penampang kaki mereka sama besar. Kedalaman jejak kaki petani C lebih dalam dari jejak kaki petani B dan jejak kaki B lebih dalam dari jejak kaki A. Perbandingan berat ketiga petani tersebut adalah petani A < petani B < petani C (C).

Penjelasan dengan langkah-langkah

Besarnya tekanan (P) dipengaruhi oleh:

Gaya tekan (F)Luas permukaan (A)

Sehingga rumus dari tekanan adalah:

P = F/A

Diketahui

Luas penampang kaki petani A, B dan C sama besar.Kedalaman jejak kaki petani C lebih dalam dari jejak kaki petani B.Kedalaman jejak kaki B lebih dalam dari jejak kaki A.

Ditanyakan

Tentukan perbandingan berat ketiga petani tersebut!

Jawab

Langkah 1

Luas penampang kaki ketiga petani sama besar, sehingga perbedaan kedalaman jejak kaki dipengaruhi oleh berat dari ketiga petani tersebut. Semakin dalam jejak kakinya, semakin besar berat dari petani tersebut.

Langkah 2

Kedalaman jejak kaki petani C lebih dalam dari jejak kaki petani B, berarti:

Berat petani C > berat petani B

atau

Berat petani B < berat petani C

Langkah 3

Kedalaman jejak kaki B lebih dalam dari jejak kaki A, berarti:

Berat petani B > berat petani A

atau

Berat petani A < berat petani B

Langkah 4

Berdasarkan langkah 2 dan 3, dapat disimpulkan bahwa perbandingan berat ketiga petani tersebut adalah:

Petani C > Petani B > Petani A

atau

Petani A < Petani B < Petani C

Saat berjalan di lumpur, itik dan ayam yang beratnya sama akan mengalami B. Jejak kaki ayam lebih dalam daripada jejak kaki itik.

[tex]\texttt{ }[/tex]

PEMBAHASAN

Marilah kita ingat kembali rumus Tekanan dan Tekanan Hidrostatis berikut ini:

[tex]\boxed{P = F \div A}[/tex]

[tex]\boxed{P_h = \rho g h}[/tex]

dimana :

P = Tekanan ( N/m² = Pascal )

F = Gaya Tekan ( Newton )

A = Luas Penampang ( m² )

Ph = Tekanan Hidrostatis ( N/m² = Pascal )

ρ = massa jenis cairan ( kg/m³ )

g = percepatan gravitasi ( m/s² )
