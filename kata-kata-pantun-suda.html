13. suda krungu panulise aksarajawa​


Nulis aksara jawa

jawaban terlampir

Pembahasan

Aksara jawa ing dhuwur nggunakake sandhangan wyanjana.

➡️ Sandhangan wyanjana ana 3 :

1. Cakra = sesulihe ra ➡️ ꦿ

→ Tuladha :

kribo = ꦏꦿꦶꦧꦺꦴtrondhol = ꦠꦿꦤ꧀ꦝꦺꦴꦭ꧀kramas = ꦏꦿꦩꦱ꧀

2. Keret = sesulihe cakra lan pepet ➡️ꦽ

→ Tuladha :

sregep = ꦱꦽꦒꦼꦥ꧀kreteg = ꦏꦽꦠꦼꦒ꧀Gresik = ꦒꦽꦱꦶꦏ꧀

3. Pengkal = sesulihe ya ➡️ ꦾ

→ Tuladha :

kyai = ꦏꦾꦲꦶkodya = ꦏꦺꦴꦢꦾSatya = ꦱꦠꦾ

➡️ Jlentrehne aksara jawa.

Suda krungu = ꦱꦸꦢꦏꦿꦸꦔꦸ

suda = aksara Sa + suku, aksara Dakrungu = aksara Ka + cakra + suku, aksara Nga + suku.

Suda krungu, nggunakake sandhangan wyanjana cakra (sesulihe ra).
