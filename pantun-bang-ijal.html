Diketahui:

Nilai siswa berdasarkan data tersebut adalah: 70 75 75 75 80 80 90 90 70 90 90 60 65 60 85 55 80 55 80 60.Jumlah siswa = 20.

Ditanya:

Berapakah rata-rata nilai dari semua siswa tersebut.?

x̄ = (x₁ + x₂ + x₃ + x₄ + x₅ + x₆ + x₇ + x₈ + x₉ + x₁₀ + x₁₁ + x₁₂ + x₁₃ + x₁₄ + x₁₅ + x₁₆ + x₁₇ + x₁₈ + x₁₉ + x₂₀ ) / n

A. 70 75 80 90 70 90 65 85 80 80 75 75 80 90 90 60 60 55 55 60

Banyak siswa yg memperoleh nilai ;

→ 55 = 2

→ 60 = 3

→ 65 = 1

→ 70 = 2

→ 75 = 3

→ 80 = 4

→ 85 = 1

→ 90 = 4

Gambar Jawaban Terlampir..

B. Nilai Rata-rata

= Jumlah nilai ÷ Banyak nilai

= (( 55 × 2 ) + ( 60 × 3 ) + ( 65 × 1 ) + ( 70 × 2 ) + ( 75 × 3 ) + ( 80 × 4 ) + ( 85 × 1 ) + ( 90 × 4 )) ÷ 20

= ( 110 + 180 + 65 + 140 + 225 + 320 + 85 + 360 ) ÷ 20

= 1485 ÷ 20

= 74,25

Jadi nilai rata-rata dari 20 siswa diatas tersebut adalah 74,25

x = rata-rata hitung

​[tex]x_{1}[/tex] = nilai sampel ke-1

n = jumlah sampel

= 2275 : 30

= 75,83

diketahui :

panjang = 16 cm

lebar = 12 cm

skala peta = 1:250

ditanyakan:

luas lahan = ... ?

penyelesaian:

P peta = Pb × skala

P peta = 16 cm × 1/250 = 0,064

L peta = 12 cm × 1/250 = 0,048

∆Luas persegi panjang= P × L

= 0,064×0,048

=0,003072 cm²

31. Jumlah kelereng rizal robi adalah 40. jika banyaknya kelereng ijal adalah 8 kelereng lebih banyak dari pada banyaknya kelereng robi. tentukan banyak nya masing" yang dimiliki oleh ijal dan robi

Robi = 16 kelereng

Rizal = 24 kelereng

Penjelasan dengan langkah-langkah:

Misalkan

Rizal = a

Robi = b

a = 8 + b

a + b = 40

8 + b + b = 40

2b = 40 - 8

2b = 32

b = 32/2

b = 16 (Kelereng Robi)

a = 8 + b

a = 8 + 16

a = 24 (Kelereng Rizal)
