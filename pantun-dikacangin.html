18. Jangan dikacangin ah aku nya


f(x) = u/v

u = 3x²
u' = 6x

v = 2x + 1
v' = 2

f'(x) = (u'.v - u.v') / v²
= [(6x).(2x + 1) - (3x²).(2)] / (2x + 1)²
= [12x² + 6x - 6x²] / (2x + 1)²
= [6x² + 6x] / (2x + 1)²
= [6x(x + 1)] / (2x + 1)²
