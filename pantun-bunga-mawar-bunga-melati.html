40. Jumlah bunga mawar dan melati di sebuah taman adalah 140 buah. Jika bungamawar dan melati mempunyai perbandingan 2 : 5 . Maka selisih bunga mawar danmelati adalah ....


Diketahui :
Jumlah mawar dan melati = 140
Perbandingan = 2 : 5

Ditanya = selisih bunga mawar dan melati

Jawab =
Selisih = 5 - 2 = 3 bagian
Selisih = (140 : (2 + 5) ) x 3
= (140 : 7 ) x 3
= 20 x 3
= 60
