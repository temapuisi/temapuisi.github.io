Diketahui :

pakaian muslim = baju koko, sarung(diskon 15%)

sandal, tas selempang (diskon 10%)

berapa uang kembalian dari Rp. 400.000

jawab

pakaian Muslim

baju koko

= 150.000- (15%× 150.000)

= 150.000 - 22.500

= 127.500

sarung

= 80.000-(15%× 80.000)

= 80.000-12.000

= 68.000

sandal

= 120.000-(10%×120.000)

= 120.000- 12.000

= 108.000

tas selempang

= 75.000- (10%×75.000)

= 75.000-7.500

= 67.500

jadi kembalian nya

= 400.000-127.500-68.000-108.000-67.500

= 29.000
