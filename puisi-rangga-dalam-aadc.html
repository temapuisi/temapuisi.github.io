15. Kelereng Adi 28 butir lebih banyak dari kelereng rangga, kelereng adi dan rangga berbanding 7 : 3 jumlah kelerang rangga adalah


Mapel Matematika
Bab Perbandingan

Adi : Rangga = 7 : 3
Adi = 28 + Rangga

Rangga = 3/7 x (28 + Rangga)
Rangga = 12 + 3/7 Rangga
4/7 Rangga = 12
Rangga = (12 x 7) / 4
Rangga = 21 kelerengmisal:
a = kelereng Adi
b = kelereng Rangga

a = 28 + b → a - b = 28
perbandingan a : b = 7 : 3
b = ?

perbandingan a - b = 7 - 3 = 4

b = (perbandingan b)/(perbandingan a - b) × (a - b)
= 3/4 × 28
= 3 × 7 = 21 butir
